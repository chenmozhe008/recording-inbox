# Changelog

## Unreleased

## 0.2.0 - 2026-07-12

- 每条录音分别生成“智能纪要”和“文字稿”两篇飞书文档，并通过一条消息提供两个入口。
- 增加 10 个内置纪要模板，默认使用通用会议纪要模板。
- 默认模型更新为 DeepSeek V4 Flash，并支持 OpenAI Chat Completions 兼容服务。
- 完成真实双文档发布与完成通知验收。

## 0.1.0 - 2026-07-11

- 默认使用已授权的 lark-cli 给当前飞书账号发送结果消息，Webhook 下沉为兼容选项。
- 增加 macOS / Windows 统一配置向导和通知权限自检。
- 修复电脑断电后遗留锁可能阻塞数小时的问题。
- 已发布纪要通知失败时只补通知，不重复创建文档。
- 增加会议、访谈、课程、项目沟通模板和自定义提示词入口。
- 增加模拟端到端、敏感信息、文档链接和跨平台 CI 检查。
- 重构安装、上传、API、安全、恢复、示例和推广文档。
- 恢复 GitHub 首页的核心优势、使用场景、产品对比和可信验证证据。
- 增加多渠道推广文案、演示录屏脚本、发布节奏和传播红线。
