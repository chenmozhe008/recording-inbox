# 移动端真机录屏清单

这份清单用于 README 教程素材，不是安装的额外步骤。每个平台录一段 30-60 秒视频即可。

## 录制前

1. 准备一条 5-10 秒、没有隐私内容的测试录音；
2. 把飞书 inbox 加入“收藏”；
3. 打开屏幕录制，关闭消息预览；
4. 不展示 API Key、文件夹权限、联系人或真实会议内容。

## iPhone

按这个顺序连续录制：

1. 在“语音备忘录”选中测试录音；
2. 分享并选择“存储到文件”；
3. 打开飞书，从“收藏”进入 inbox；
4. 点 `+` 或“上传”，从“文件”选中录音；
5. 等列表中出现文件名，停留 2 秒后结束录屏。

建议文件名：`iphone-upload-to-feishu.mp4`。

## Android

按这个顺序连续录制：

1. 打开飞书，从“收藏”进入 inbox；
2. 点 `+` 或“上传”；
3. 从系统文件管理器选择测试录音；
4. 等列表中出现文件名，停留 2 秒后结束录屏。

建议文件名：`android-upload-to-feishu.mp4`。

## 验收

- 录屏能看清“收藏 -> inbox -> 上传 -> 文件出现”的完整闭环；
- 没有剪掉文件真正出现在 inbox 的一刻；
- 没有账号、联系人、真实录音或密钥泄露；
- 当前飞书版本按钮名称不同不算问题，教程文字按录屏同步更新即可。
