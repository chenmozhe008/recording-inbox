# 维护者：移动端真机录屏清单

> 仅供维护者更新 README、教程截图或演示视频时使用，不是普通用户的安装或上传步骤。

每个平台录一段 30-60 秒视频即可。

## 录制前

1. 准备一条 5-10 秒、没有隐私内容的测试录音；
2. 确认飞书云盘中已经有“录音收件箱”文件夹；
3. 打开屏幕录制，关闭消息预览；
4. 不展示 API Key、文件夹权限、联系人或真实会议内容。

## iPhone

按这个顺序连续录制：

1. 在“语音备忘录”选中测试录音；
2. 分享并选择“存储到文件”；
3. 打开飞书，进入“云盘”（部分版本显示“云文档”）；
4. 找到并打开“录音收件箱”文件夹；
5. 点击右下角“上传”（部分版本显示 `+`），选择“上传文件”；
6. 从“文件”中选中录音；
7. 等列表中出现文件名，停留 2 秒后结束录屏。

建议文件名：`iphone-upload-to-feishu.mp4`。

## Android

按这个顺序连续录制：

1. 打开飞书，进入“云盘”（部分版本显示“云文档”）；
2. 找到并打开“录音收件箱”文件夹；
3. 点击右下角“上传”（部分版本显示 `+`），选择“上传文件”；
4. 从系统文件管理器选择测试录音；
5. 等列表中出现文件名，停留 2 秒后结束录屏。

建议文件名：`android-upload-to-feishu.mp4`。

## 验收

- 录屏能看清“云盘 -> 录音收件箱 -> 右下角上传 -> 选择文件 -> 文件出现”的完整闭环；
- 没有剪掉文件真正出现在“录音收件箱”的一刻；
- 没有账号、联系人、真实录音或密钥泄露；
- 当前飞书版本按钮名称不同不算问题，教程文字按录屏同步更新即可。
- 入口显示为“云文档”或“云盘”都可以，但必须进入同一个“录音收件箱”文件夹。
