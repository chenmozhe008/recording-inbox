# 上传录音：iPhone、Android 和电脑

所有入口最终都把录音上传到飞书云盘里的“录音收件箱”文件夹。

> 飞书移动端会随版本更新调整按钮名称和位置。本文给出的是稳定的操作逻辑：进入“录音收件箱”后，选择“上传文件”即可。

不同版本的入口可能显示为“云文档”或“云盘”，右下角的上传入口也可能显示为 `+`。它们都是正常的版本差异。上传录音时选择“上传文件”，不要选择“导入为在线文档”。

## 电脑拖拽：最简单

适合微信、网盘、聊天软件或已经同步到电脑的音频：

1. 打开飞书云盘中的“录音收件箱”；
2. 把音频拖进窗口；
3. 等电脑下一轮自动接手。

支持 `.wav`、`.mp3`、`.m4a`、`.aac`、`.mp4`、`.mov`。

## iPhone

系统“语音备忘录 → 分享 → 飞书”在部分版本只提供聊天或飞书妙记，不一定能保存到云盘“录音收件箱”。最稳定的通用路径是：

1. 语音备忘录录完，点分享；
2. 选择“存储到文件”；
3. 打开飞书 App，进入“云盘”（部分版本显示“云文档”）；
4. 找到并打开“录音收件箱”文件夹；
5. 点击右下角“上传”（部分版本显示 `+`），选择“上传文件”；
6. 在“文件”中选择刚才保存的录音。

看到录音文件出现在“录音收件箱”中，就表示上传成功。第一次找到“录音收件箱”后，可以把这个文件夹加入“收藏”；收藏只是以后少找一层目录，不是上传的必需步骤。

## Android

不同品牌的系统录音机分享菜单差异很大。通用路径：

1. 打开飞书 App；
2. 进入“云盘”（部分版本显示“云文档”）；
3. 找到并打开“录音收件箱”文件夹；
4. 点击右下角“上传”（部分版本显示 `+`），选择“上传文件”；
5. 从系统文件管理器选择录音。

第一次找到“录音收件箱”后，可以把它加入“收藏”，方便下次直接进入。

如果文件管理器的分享菜单明确出现“保存到云盘”，也可以使用；不要选择“音视频转文字”，那会走飞书妙记并消耗对应额度。

## iPhone 快捷指令（高级）

适合每天大量录音、愿意维护飞书自建应用的人。它不是首次安装必选项。

大致流程：

1. 飞书开放平台创建自建应用并开启文件上传权限；
2. “录音收件箱”文件夹共享给应用；
3. 快捷指令接收语音备忘录分享输入；
4. 获取 tenant access token；
5. 调用飞书文件上传 API 写入“录音收件箱”；
6. 本地显示上传成功。

App Secret 会保存在快捷指令中，不要公开分享该快捷指令、截图或泄露 Secret。飞书开放平台权限和快捷指令界面会变化；失败时退回飞书 App 手动上传，不影响主流程。

## 电脑没开机怎么办？

- 上传仍然成功，录音留在飞书“录音收件箱”；
- 电脑开机后自动下载和处理；
- 电脑关机期间本地程序无法发送“电脑离线”消息；
- 急用时可以临时用飞书妙记处理这一条。

## 如何判断上传成功？

在飞书“录音收件箱”能看到音频即可。电脑在线时，通常下一分钟接手；电脑接收到原音频后，会移进“已接收原始音频”子文件夹。
